Project.java основной, в нём происходит инициализация объекта класса Calendar и вызываются его методы
Класс Calendar использует библиотеку java.time
Календаорь хранится в виде двумерного массива 6*7 состоящего из объектов класса Days, который по факту несет в себе
числовое поле для хранения числа в месяце и поле типа String, в котором хранится цвет, которым это число отрисовывать при выведении
